Corrida Maluca

O projeto é um jogo de corrida onde dois jogadores competem para chegar primeiro à linha de chegada. A movimentação de cada jogador é aleatória, controlada pelas teclas "a" (Jogador 1) e "s" (Jogador 2). O fundo da tela muda dependendo se o jogo está em foco. Quando um jogador ultrapassa a linha de chegada, o jogo exibe o vencedor e para automaticamente.

Palavra Interativa

O projeto exibe uma palavra de forma gradual, onde as letras são reveladas à medida que o usuário move o mouse na horizontal. A palavra exibida é escolhida aleatoriamente de uma lista e, conforme o mouse se move pela tela, mais letras dessa palavra vão aparecendo. O texto é centralizado e o fundo é branco, com as letras aparecendo em preto.

Arte Interativa

O projeto desenha um rosto simples, com olhos, nariz, boca e sobrancelhas. As pupilas dos olhos seguem o movimento do mouse, criando uma interação dinâmica. Quando o mouse é pressionado, as coordenadas do cursor são registradas no console. O fundo escuro e as formas claras do rosto fazem um contraste visual, destacando os elementos. As pupilas se ajustam suavemente à posição do mouse, criando um efeito de "olhar" mais realista.
